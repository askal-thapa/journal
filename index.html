The PWA Project

This Progressive Web App serves as a live portfolio and learning log for the FGCT6021 Mobile Application Development unit. It documents my journey through modern web technologies, showcases my technical projects, and demonstrates fundamental front-end skills like mobile-first responsive design, progressive enhancement, and user experience optimization.

Built with modern web standards including CSS Grid, Flexbox, ES6+ JavaScript, and PWA APIs. Features include dark mode, offline functionality, responsive design, and smooth animations for an enhanced user experience.

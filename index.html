The PWA Project

A learning journal and portfolio, built lab by lab.

Project Overview

This Progressive Web App serves as a live portfolio and learning log for the FGCT6021 Mobile Application Development unit. It documents my journey through modern web technologies and demonstrates front-end excellence.

Key Features

  • Mobile-first responsive layout
  • Light and dark themes
  • Installable PWA that works offline
  • Flask backend with a JSON reflections API
  • Keyboard and screen-reader friendly

Technologies Used

  • Frontend: HTML5, CSS3, ES6+ JavaScript
  • Design: CSS Grid, Flexbox, Custom Properties
  • Backend: Python, JSON file storage
  • PWA: Service Workers, Web Manifest

Course Progress

All seven labs completed — from HTML foundations to a working PWA

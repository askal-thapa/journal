Week 2: Frontend Fundamentals (HTML/CSS)3 min read

Posted: October 27, 2025

Reflection Questions

  1. Approach to Mobile-First Design: Started with narrow layout and used `min-width` media queries to progressively enhance for larger screens.
  2. Most Useful Concept: Flexbox for aligning complex navigation elements and creating responsive layouts without complex float-based solutions.

  • Mobile-first design leads to better performance
  • CSS Grid and Flexbox complement each other perfectly
  • Semantic HTML improves accessibility and SEO

Week 3: JavaScript Basics & Theme Toggle4 min read

Posted: November 2, 2025

What I learned

  • DOM manipulation: Selecting elements with querySelector, changing innerHTML, and toggling classes dynamically.
  • Event handling: Adding click listeners to respond to user actions and creating interactive experiences.
  • localStorage: Saving user preferences so choices persist across browser sessions and page reloads.
  • Progressive enhancement: Injecting shared navigation from one JS file to keep HTML DRY and maintainable.

What I built

I used JavaScript to add a sophisticated dark/light theme toggle system. The script dynamically injects navigation into each page, renders an elegant theme button, toggles a dark-mode class on the body, and saves the user's choice to localStorage. The implementation includes smooth transitions and accessibility features.

Code Implementation

The JavaScript is modular and well-structured, with separate functions for navigation loading and theme switching. Event listeners are properly bound, and the code follows modern ES6+ practices.

Reflection

This task demonstrated how a few well-crafted lines of JavaScript can dramatically enhance user experience. It reinforced the importance of accessibility (aria-labels), progressive enhancement, and state persistence in modern web development.
